Overview (English)

The Blockchain Security Analyzer is a Python-based tool, built with the Flask framework, designed for static analysis of Solidity smart contracts. It identifies common security vulnerability patterns in contracts, helping developers build more secure decentralized applications (dApps).

Developed by Gabriel Demetrios Lafis.

Features (English)

Static Analysis

Identifies common security vulnerabilities in Solidity smart contracts.

RESTful API

Provides a simple API to submit contract code and receive analysis reports.

Reentrancy Detection

Detects patterns of external calls that could lead to reentrancy attacks.

Integer Overflow/Underflow Detection

Identifies arithmetic operations that can result in integer overflow or underflow.

Timestamp Dependency Detection

Flags the use of block.timestamp, which can be manipulated by miners.

Hardcoded Gas Limit Detection

Warns about the use of fixed gas limits that can cause transaction failures.
