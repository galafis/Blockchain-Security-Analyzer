Overview

O Blockchain Security Analyzer é uma ferramenta baseada em Python, construída com o framework Flask, projetada para análise estática de contratos inteligentes Solidity. Ele identifica padrões comuns de vulnerabilidades de segurança em contratos, ajudando desenvolvedores a construir aplicações descentralizadas (dApps) mais seguras.

Desenvolvido por Gabriel Demetrios Lafis.

Features

Análise Estática

Identifica vulnerabilidades de segurança comuns em contratos inteligentes Solidity.

API RESTful

Fornece uma API simples para submeter o código do contrato e receber relatórios de análise.

Detecção de Reentrancy

Detecta padrões de chamadas externas que podem levar a ataques de reentrancy.

Detecção de Integer Overflow/Underflow

Identifica operações aritméticas que podem resultar em overflow ou underflow de inteiros.

Detecção de Timestamp Dependency

Sinaliza o uso de block.timestamp, que pode ser manipulado por mineradores.

Detecção de Hardcoded Gas Limit

Alerta sobre o uso de limites de gás fixos que podem causar falhas em transações.

API Endpoints

A API do Blockchain Security Analyzer oferece os seguintes endpoints:

  • GET /: Retorna informações gerais sobre o projeto e os endpoints disponíveis.
  • GET /api/status: Retorna o status atual da API.
  • POST /api/analyze: Aceita o código-fonte de um contrato Solidity (JSON com a chave code) e retorna um array de vulnerabilidades detectadas.

Para mais detalhes sobre como usar a API, consulte a documentação no GitHub.
